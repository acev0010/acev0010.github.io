Five tags I learned

<ol> Ordered list
This tag helps us makes a list where the order is important.
<ul> Unordered list
This tag helps us makes a list where the order is not important.
<main> Main
This tag indicates the starting main content of the page.
<em> Emphasis
This tag emphasizes a slightly more important text.
<sub> Subscript
This tag makes the selected text a subscript.
